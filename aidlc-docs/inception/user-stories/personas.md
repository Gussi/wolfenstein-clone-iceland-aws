# User Personas — "Pots & Parliament"

## Persona 1: The Player (Primary)

| Attribute | Description |
|-----------|-------------|
| **Name** | "Jón/Jóna Almennur" (the everyman) |
| **Role** | The in-game protagonist AND the real-world player |
| **Motivation** | Wants fast, satisfying arcade action with comedic payoff |
| **Frustration** | Unclear feedback, sluggish controls, unfair difficulty |
| **Expectation** | Immediate responsiveness, clear hit feedback, short retry loops |
| **Context** | Playing in a web browser, expects instant load, keyboard+mouse |

### Key Needs
- Responsive movement that feels fast and arcade-like
- Clear visual and audio feedback for every action
- Short death-to-retry loop (no long loading screens)
- Discoverable mechanics (intuitive controls)
- Comedic tone maintained throughout — losing should feel funny, not punishing

## Persona 2: The Explorer

| Attribute | Description |
|-----------|-------------|
| **Name** | Subset of Player persona focused on discovery |
| **Role** | Player who seeks secrets, reads environmental details |
| **Motivation** | Find push-wall secrets, appreciate satirical set dressing |
| **Frustration** | Missing secrets with no indication they exist, bland environments |
| **Expectation** | Visual hints for secrets, rewarding discovery, environmental humor |

### Key Needs
- Subtle visual cues that secrets exist (slightly different wall texture)
- Satisfying audio/visual feedback on secret discovery
- Reward for exploration (health pickups, score)
- Environmental storytelling through textures and room layouts

## Persona 3: The Arcade Completionist

| Attribute | Description |
|-----------|-------------|
| **Name** | Subset of Player persona focused on mastery |
| **Role** | Player who wants to clear all enemies, find all secrets, maximize score |
| **Motivation** | 100% completion, high "connections" score |
| **Frustration** | Not knowing how many enemies/secrets remain |
| **Expectation** | End-of-level stats, clear enemy counts, discoverable score mechanics |

### Key Needs
- Level completion summary (enemies dispersed, secrets found, time)
- Clear indication of remaining enemies
- Score counter visible during gameplay
